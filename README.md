A fullstack Airbnb clone built with MERN (MongoDB, Express, React, Node.js). Learn how to create a booking app with user accounts, property listings, and bookings. Perfect for beginners and portfolio projects.

 Features

User Authentication: Sign up, login, logout
Account Management: Update profile info
Property Listings: Add, edit, delete accommodations
Photo Uploads: Via link or device
Search & Browse: Filter listings by location and features
Booking System: Book properties and view bookings
Responsive UI: Works on desktop & mobile

 Tech Stack

Frontend: React.js, Tailwind CSS / Bootstrap
Backend: Node.js, Express.js
Database: MongoDB & Mongoose
Authentication: JWT & bcrypt
Image Upload: Cloudinary or local
Payment Integration: Optional (Razorpay / Stripe)

Setup

Backend
cd backend
npm install
# Add .env with PORT, MONGO_URI, JWT_SECRET, EMAIL_USER, EMAIL_PASS
npm run dev

Frontend
cd frontend
npm install
npm start
